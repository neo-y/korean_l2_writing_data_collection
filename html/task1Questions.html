Translation Task instruction


Please read the instruction below carefully before you start!


  • In this task, you are asked to translate 40 English sentences to Korean.
  • There is no time limit, but it is recommended to finish all sentences within 60 minutes.
  • Give yourself about 1~1.5 minute for each sentence.
  • You can use dictionary, but DO NOT use machine translator.
  • Do not get help from other people.
  • Please write only Korean alphabet and punctuation in the answer fields
  • If you cannot translate a sentence at all, please write only 'X' in the answer field. If you leave any answer field empty, the submission will be not done.
  • Please note that your answer will be not saved if you leave this page.
  • If you are done, please click 'submit' button below.
  • Your answer will be saved anonymously.
  • Before you start, please give your User ID and five digit access code for the task1 again.
