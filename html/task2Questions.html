Writing Task instruction


Please read the instruction below carefully before you start!


  • In this task, you are asked to answer five questions by telling your own stories.
  • There is no time limit, but it is recommended to finish the task within 75 minutes.
  • Give yourself about 15 minutes for each question.
  • For each question, write 100-200 characters. (ex. 안녕하세요 = 5 characters)
  • You can use dictionary, but DO NOT use machine translator.
  • Do not get help from other people.
  • Please write only Korean alphabet and punctuation in the answer fields
  • Please note that your answer will be not saved if you leave this page.
  • If you are done, please click 'submit' button below.
  • Your answer will be saved anonymously.
  • Before you start, please give your User ID and five digit access code for the task2 again.
